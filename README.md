# Descubriendo Lugares de Argentina

Este es un blog de turismo dedicado a difundir paisajes, cultura y rincones únicos de Argentina.

## Propósito del Proyecto
El sitio fue desarrollado como entrega para el curso de Front-End, aplicando los siguientes conocimientos técnicos:
- Estructura semántica de HTML5.
- Maquetación responsiva con Flexbox (Sección Lugares) y CSS Grid (Sección Reseñas).
- Estilos avanzados e interactividad mediante Pseudo-clases y Transiciones en CSS3.
- Media Queries para adaptabilidad en dispositivos móviles.
- Integración de formularios funcionales con Formspree.
